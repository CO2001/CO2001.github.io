Christian's Android Project!

The App that my partner and I created is called Guess the Color. In this game we will give you a riddle and some colors, you have to figure out the riddle and guess the color correctly. The app has 3 settings for the colors, 3 colors, 5 colors, and 7 colors. The riddles get harder with each level. If you get the answer correct, the correct screen should be shown but with a glitch that we canâ€™t seem to figure out, it crashes the game. But if you get the answer wrong the incorrect screen will show.

A problem that my partner and I came across was that every time a new screen was opened the main menu music kept looping over and over again. So, what we did was, we removed the music player from each screen. That made it so that the music would only replay on the main menu screen. So to fix that we unchecked the looping option so that after the music ended it would just be silent. I learned that making an app is a lot harder than I first thought. The process of making this app was challenging but fun. I have a rough understanding of how to make an app on the app inventor site and I had fun working on this project.
